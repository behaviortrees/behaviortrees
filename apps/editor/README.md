# Behavior Tree Editor (React)

The next-generation editor for [behaviortrees.com](https://www.behaviortrees.com), live at [new.behaviortrees.com](https://new.behaviortrees.com). A ground-up modernization of the classic editor built with React 19, TypeScript, Zustand, and React Flow, reading and writing the same behavior3 JSON format — projects round-trip cleanly between both editors.

## Development

```sh
pnpm install
pnpm dev        # dev server
pnpm test       # vitest
pnpm build      # type-check + production build
pnpm lint
```
